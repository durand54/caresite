System Requirements:

There are a wide variety of hardware and software configurations in the marketplace. The minimum requirements as set forth herein are based on achieving the broadest possible range of compatibility:

Macintosh:
  • Mac OS 8.6, 9.x (9.1 and later recommended)
  • Mac OS X (10.1.x recommended)
  • G3, G4 or faster processor
  • 128MB of RAM
  • 100 MB hard drive


NOTE: You may run into a published document that indicates a browser or OS may require less RAM that is noted above. It may be possible to run a browser on the Mac with less memory than specified. However, it is unlikely that consistent performance will be possible with less than 128MB of RAM. As an example, Mac OS 10.2 requires 128MB of RAM to run.

Windows PC:

  • Windows, NT, 2000, XP
  • Pentium II 400Mhz or faster processor
  • 80MB hard drive


NOTE: Minimum requirements for Windows-based PC systems will vary depending on the compliment of hardware, software and applications in use. Be aware that Netscape Navigator version 7 will also run under the Windows configuration listed herein while running under RedHat Linux, version 6.2,7.0, 7.1, 7.2 or higher.

Connection:

The application can run using a dial-up modem, DSL, cable, or T-1 connection.
  • Macintosh - 56Kbps or faster
  • Windows - 56Kbps or faster


NOTE: If a user is trying to connect using a dial-up modem, please check to ensure that they are connecting at an appropriate speed. Most dial-up farms will redirect an incoming connection to a slower modem if there is too much traffic to manage the higher bandwidth. The application will run, but very slowly on connections slower than 44.1Kbps.
